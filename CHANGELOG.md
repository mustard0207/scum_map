# 更新日志

## v7.6 (2026-06-26)

### 新增
- **Z3+Z4+Z6 瓦片分层加载**：根据缩放级别动态切换瓦片层级
  - Z2 (scale < 1.0)：本地 JPG 底图
  - Z3 (1.0 ~ 1.5)：jsDelivr 网络 webp，2×2 网格
  - Z4 (1.5 ~ 3.0)：jsDelivr 网络 webp，4×4 网格
  - Z6 (scale ≥ 3.0)：jsDelivr 网络 webp，16×16 网格，Z4 作为兜底
- **缩放拖拉条**：右侧垂直拖拉条，以十字准星为中心缩放，支持鼠标和手指操作
- **本地缓存机制**：网络瓦片首次加载后自动缓存到用户本地，后续直接读取

### 优化
- 修复 `_clampOffset` 时序 bug（scale 先更新再钳制边界）
- 坐标面板与全局视角按钮位置互换
- 坐标标签从"经度/纬度"改为"X/Y"，与游戏坐标格式一致
- 地图区域使用动态 top/bottom 定位，消除底栏黑边
- 移除关于页面和 scum-map.com 数据来源引用
- 添加 debug 日志和 `wx.getWindowInfo` 防御
